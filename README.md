## FlowZone
FlowZone is your AI-powered Whiteboard.

## Getting Started
1) Create .env file from .env.example and enter Gemini API Key 
2) Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
